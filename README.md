# Air Quality Analysis Dashboard

## Project Overview
This project analyzes air quality data using Excel, SQL, and Power BI. It includes data cleaning, SQL analysis, and an interactive Power BI dashboard to identify pollution trends and generate business insights.

## Tools Used
- Microsoft Excel
- SQL (MySQL)
- Power BI

## Project Structure

### Excel-Air-Quality-Dashboard
- Data Cleaning
- Excel Dashboard
- Dashboard Screenshot

### SQL-Air-Quality-Analysis
- SQL Queries
- Data Analysis
- SQL Practice Questions

### PowerBI-Air-Quality-Dashboard
- Power BI Dashboard (.pbix)
- Dashboard Screenshot

## Key Insights
- Identified the most polluted cities.
- Compared AQI across countries.
- Analyzed PM2.5 and PM10 levels.
- Built interactive dashboards using Power BI.
